The Virtual Toy Store is a web-based e-commerce application built using the Django framework. The platform allows customers to browse,
search, and purchase toys online through a user-friendly interface. It includes features for product management,shopping cart functionality,
secure user authentication, and an admin dashboard for store management.
The main goal of the Virtual Toy Store is to demonstrate how a full-stack e-commerce website can be built using Django, showcasing user 
authentication, database operations, template rendering, and admin control. It is suitable as a college project, portfolio project, 
or learning project for web development.
Home page showcasing featured toys, categories, and promotions
Product catalog with filtering by age group, category, and price
Product detail pages with images, descriptions, and stock availability
User registration and login using Django authentication
Shopping cart to add, update, or remove items
Checkout system (cash-on-delivery or dummy payment flow)
Order history for logged-in customers
Admin dashboard for managing products, categories, and inventory
Add/Edit/Delete products with images
Order management (view customer orders and update status)
